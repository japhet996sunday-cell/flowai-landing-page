# FlowAI

A premium, production-ready AI SaaS landing page built using only HTML5, CSS3, and Vanilla JavaScript.

## Features

- Responsive design
- Sticky navigation
- Mobile menu
- Hero section
- Features section
- Pricing cards
- Testimonials
- FAQ accordion
- Animated counters
- Scroll reveal animations
- Dark/Light mode
- Smooth scrolling
- Back-to-top button
- Contact form validation
- Toast notifications

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
